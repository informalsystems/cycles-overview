# Cycles Notebooks

These python notebooks contain some examples of applications that can be built
with the Cycles model.

- [Four Ways To Settle](./four_ways_to_settle.ipynb) explores the fundamentals
  of settlement via double-entry bookkeeping
- [P2P or Pay](./p2p_or_pay.ipynb) explores the opportunity for p2p credit
  extension in a trade network
- [Discount Ladder](./discount_ladder.ipynb) explores a unique application made
  possible by Cycles, allowing for invoice discounting within the graph
- [Limit Order DeX](./limit_order_multi_DEX.ipynb) shows how to design a new
  kind of DeX with the Cycles architecture
